# Template Package

这是一个可直接复制到业务仓库里的示例包。

它保留了一个 WMS 风格的例子，目的是让你更容易看懂复杂业务项目中的 Skill 组织方式，而不是要求你照抄这些名字。

迁移到自己的仓库时，优先改这些位置：

1. `AGENTS.md`
2. `docs/guides/AI协作研发章程.md`
3. `docs/prompts/00-department-standards.md`
4. `docs/skills-src/manifest.yaml`
5. `.github/copilot-instructions.md` 和 `.claude/skills/` 的生成结果
