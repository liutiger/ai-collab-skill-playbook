# Workflow

## 边界定义

- 起点：接口、MQ、Job、事件或手工操作
- 终点：DB 状态、消息外发、外部系统、返回值
- 范围：主链路还是含异常分支

## 真实链路

1. 入口类 / 方法
2. 编排 Service
3. 领域逻辑 / 策略 / 工厂
4. DAO / Mapper / Entity
5. MQ / Feign / Redis / 第三方系统

## 关键控制点

- 分支
- 幂等
- 并发 / 锁
- 事务
- 重试 / 补偿 / 回滚

## 关键副作用点

- DB 写入
- MQ 外发
- Feign / HTTP 调用
- 缓存更新
- 状态推进
